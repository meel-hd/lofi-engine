
# Contributing

We welcome contributions from the community!

Check the [issue tracker](https://github.com/meel-hd/lofi-engine/issues) if you want to help, report a bug, or discuss new ideas.

If you're interested in contributing to LoFi Engine, please follow these steps:

1. Fork the repository.
2. Create a new branch for your feature or bugfix.
3. Implement and test your fix or feature. 
4. Make your changes and commit them with clear and descriptive messages.
5. Push your changes to your fork.
6. Open a Pull Request to the `master` branch of this repository.
